HTML Common Tag Cheat Sheet

Common Tag Table

Made by: Jere Kukkohovi <08042023>

Tag Name Description
<!DOCTYPE HTML> Document type tag. Not really a tag, however tells the browser the type of file that will be loaded and rendered.
<html> </html> Html opening and closing tags. Tags used to specify the beginning and end of an HTML document
<head></head> Head opening and closing tags. Tags used for general information about the website. For example metadata, links to css file, fonts and the title for the browser tab.
<title></title> Title opening and closing tags. Declares the title for the webpage, renders the typed content inside it in the browser tab when placed in the head of the document.
<body></body> Body opening and closing tags. Content of the html document is placed in between these tags, for example; text, tables and media.
<h1-h6></h1-h6> Heading tags from h1 to h6 Heading tags for document from highest(biggest) in hierarchy to smallest. For example <h1>Heading</h1> displays a largest sized Heading in the browser.
<p></p> Paragraph opening and closing tags. Typing anything between the paragraph, displays text in the body of the document. Smaller text and used below the headings.
<b></b> Bold tags. Used to render any text in the browser as bold text.
<i></i> Italic tags. Used to render any text in the browser as italic text.
<u></u> Underlining tags. Used to render any text in the browser as underlined text.
<center></center> Centering tags. Text between the center tags is used to center it in the html document.
<span></span> Span tags. Span tag is an inline container that is used to mark up a part of text or a part of the document. It is easily styleable with CSS or manipulated with JavaScript.
<div></div> Division opening and closing tags. Division tag or div is a tag that allows to group different tags in an html document. It can be given a class to add external styling with CSS.
<em></em> Emphasis tags. Emphasis can be used to emphasize certain words in HTML document. Content in between the tags will appear slanted or italicized.
<sup></sup> Superscript tags. Superscript allows the enclosed text to be above the rest. For example wanting a number squared(102), cubed (103) or TM for trademarkTM tag.
<sub></sub> Subscript tags. Subscript is the opposite of the superscript tags. Text enclosed in it will appear lower than other text. For example if wanting a chemical formula for H2O
<br> Break tag. Break or br is a self-closing tag, it is used to start all the content after it on a new line.
<ol></ol><li></li> Ordered list and list opening and closing tags. Ordered list tag in combination with a list tag makes a numbered list.
<img src=""/> Image tag. Self closing image tag, used to link images or logos on the website, with src part being the source.
